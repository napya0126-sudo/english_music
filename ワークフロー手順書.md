# ワークフロー手順書
## J-pop英語化・AIボーカル生成プロジェクト

*対象環境：M4 Mac Mini / Applio（RVC）/ audio-separator / Audacity*

---

## 全体の流れ

```
[原曲] → [伴奏分離] → [英語歌詞作成] → [声モデル学習] → [ガイドボーカル]
                                                              ↓
                                      [RVC変換] → [AIボーカル] → [Audacityでミックス]
                                                                       ↓
                                                              [完成MP3] → [Android転送]
```

---

## STEP 1｜原曲の準備

### 1-1. 音源ファイルの形式
- 推奨フォーマット：WAV（44.1kHz, 16bit以上）または FLAC
- MP3は分離品質が落ちる場合あり。CD取り込みはWAVで
- ファイルを `songs/originals/` に配置

```bash
# 例：CDをリッピングしたWAVファイルを配置
songs/originals/
  └── bump_of_chicken_天体観測.wav
```

---

## STEP 2｜ボーカル分離

### 2-1. audio-separator を使う

```bash
# 音声分離モードを起動（エイリアス設定後）
separate-vocals

# ボーカル分離を実行（MDX23Cモデル推奨）
audio-separator songs/originals/曲名.wav \
  --model_filename MDX23C-8KFFT-InstVoc_HQ.onnx \
  --output_dir songs/ \
  --output_format WAV

# 出力されるファイル：
# songs/曲名_(Instrumental)MDX23C-8KFFT-InstVoc_HQ.wav  ← 伴奏
# songs/曲名_(Vocals)MDX23C-8KFFT-InstVoc_HQ.wav        ← 原曲ボーカル
```

### 2-2. 出力ファイルの整理

```bash
# ファイルをそれぞれのフォルダに移動
mv songs/*Instrumental*.wav songs/instrumentals/
mv songs/*Vocals*.wav songs/vocals_original/
```

### 2-3. 品質チェック
- Audacity で伴奏ファイルを開き、ボーカル漏れがないか確認
- 漏れがひどい場合は別モデル（`htdemucs_6s` など）を試す

---

## STEP 3｜英語歌詞の作成

### 3-1. 日本語原曲歌詞を入手
- `lyrics/japanese/曲名.txt` に保存

### 3-2. Claude で英語化（推奨フロー）
Claudeに以下のように依頼する：

```
以下の日本語歌詞を英語に翻訳してください。
条件：
- 原曲の世界観・感情を最優先（直訳不要）
- 各行の音節数を日本語歌詞に合わせてください
- 英語として自然な発音・アクセントになるように調整
- 英語歌詞の隣に音節数を記載してください

[ここに日本語歌詞を貼る]
```

### 3-3. 音節確認のコツ
- 強調される拍（ビート）に英語の強音節が来るようにする
- 1音節 = 1拍を基本として数える
- 例：「ten-ta-i-kan-so-ku」= 6音節 → 英語も6音節以内に

### 3-4. 保存
- `lyrics/english/曲名_lyrics.txt` に保存

---

## STEP 4｜自分の声モデルの学習（Applio）

### 4-1. 音声データの録音

| 項目 | 推奨値 |
|------|--------|
| マイク | DJI Mic（ゲイン：低め） |
| 録音時間 | 15〜20分（多いほど品質向上） |
| 内容 | 本の朗読、詩の音読、歌詞の読み上げ等 |
| 環境 | 無音の部屋、クローゼット内が理想 |
| 形式 | WAV / 44.1kHz以上 |

- 録音ファイルを `voice_model/recordings/` に配置

### 4-2. Applio を起動

```bash
# Applio WebUI 起動（エイリアス設定後）
start-applio

# → ブラウザで http://localhost:7860 が開く
```

### 4-3. モデル学習の設定（Applio WebUI）

```
Tab: Training
  Model Name    : [自分の名前やニックネーム]
  Dataset Path  : ~/English_music/voice_model/recordings/
  Epochs        : 200〜300（最初は100でテスト）
  Batch Size    : 4（M4 Mac Miniは8まで可）
  Sample Rate   : 40000 または 48000
```

- 学習完了後、`voice_model/models/` にモデルを保存

---

## STEP 5｜ガイドボーカルの作成

### 方法A：自分でざっくり歌う（推奨・最初はこちら）

1. カラオケトラック（STEP 2で分離した伴奏）を再生しながら録音
2. 英語歌詞をできるだけメロディに合わせて歌う（音程が外れても良い）
3. `guide_vocals/曲名_guide.wav` として保存

### 方法B：AI TTS でガイドを生成（発展）

ElevenLabsを使う場合（有料）：
- サイトにアクセス → Text to Speech
- 英語歌詞を入力し、落ち着いた声で生成
- 音声をダウンロード → `guide_vocals/` に保存

StyleTTS2を使う場合（無料・ローカル）：
- 別途インストールが必要（後述の発展編を参照）

---

## STEP 6｜ガイドボーカルをAI変換（Applio）

### 6-1. 変換設定（Applio WebUI）

```
Tab: Inference
  Model File    : voice_model/models/[自分のモデル].pth
  Index File    : voice_model/models/[自分のモデル].index
  Input Audio   : guide_vocals/曲名_guide.wav
  Pitch (f0)    : 0（ガイドに合わせる）→ 必要に応じて±2〜4調整
  Method        : rmvpe（高精度・推奨）
  Index Rate    : 0.7〜0.9（高いほど自分の声に近い）
```

### 6-2. 変換実行 → 出力確認
- 出力されたAIボーカルを試聴し、音程・声質を確認
- ピッチが合わない場合は Pitch 値を微調整

---

## STEP 7｜Audacity でミックス

### 7-1. プロジェクトを開く

1. Audacity を起動
2. `ファイル → インポート → 音声` で伴奏トラックをインポート
3. 同様にAIボーカルトラックをインポート

### 7-2. タイミング合わせ

- AIボーカルトラックを選択 → タイムシフトツール（F5）でズレを調整
- 波形の目視＋再生で確認
- 必要に応じてキャプチャーツールで細部を揃える

### 7-3. エフェクト処理（推奨順序）

```
AIボーカルトラックに適用：
1. 音量正規化：エフェクト → 正規化 → -1dB
2. EQ：低音域（200Hz以下）を少し削って抜けを良くする
3. リバーブ：エフェクト → リバーブ → Room Size 20〜30%
4. コンプレッサー：ダイナミクスを整える
```

### 7-4. エクスポート

```
ファイル → エクスポート → MP3でエクスポート
  ビットレート : 320kbps（高音質推奨）
  保存先       : songs/output/曲名_english.mp3
```

## STEP 8｜Android転送

### 方法1：Google Driveで同期（推奨）
```bash
# Google Drive に songs/output/ フォルダを同期
# → Androidの Google Drive アプリ → オフライン利用可能に設定
```

### 方法2：USB接続で直接転送
```
Mac ↔ Android を USB接続
→ Android を「ファイル転送モード」に設定
→ Finder でAndroidが見える → 音楽フォルダにドラッグ
```

### Android 再生アプリ（推奨）
- **PowerAMP**：高音質、バックグラウンド再生◎
- **Musicolet**：無料、シンプル
- **Neutron Music Player**：音質重視派向け

## トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| ボーカル分離後にボーカルが残る | モデルが楽器と混同 | `htdemucs_6s` モデルを試す |
| Applio が起動しない | Python環境の問題 | `source ~/.venvs/applio/bin/activate` を確認 |
| AIボーカルの音程がズレる | Pitch設定の問題 | rmvpeメソッドで ±1〜2 ずつ調整 |
| 声が金属的・不自然 | Index Rateが高すぎ | Index Rate を 0.5 に下げる |
| Audacityでタイミングがズレる | ガイドの拍がブレている | ガイドを録り直す or 細かく手動調整 |
